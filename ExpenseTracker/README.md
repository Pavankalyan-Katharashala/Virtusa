# 💰 Smart Expense Tracker (CLI - Python)

## 📌 Project Overview

This project is a **Command Line Interface (CLI)-based Smart Expense Tracker** developed using Python. It helps users record, categorize, and analyze their daily expenses efficiently. The system provides useful insights into spending patterns and highlights areas where users can reduce expenses.

---

## 🎯 Objectives

* Record daily expenses (date, category, amount, description)
* Categorize expenses (Food, Travel, Bills, etc.)
* Generate monthly summaries
* Analyze spending patterns

---

## 🚀 Features

* ➤ Add new expenses
* ➤ View all recorded expenses
* ➤ Monthly expense summary
* ➤ Category-wise expense analysis
* ➤ Identify highest spending category
* ➤ Suggest areas to reduce spending
* ➤ Pie chart visualization using matplotlib

---

## 🛠️ Technologies Used

* Python
* CSV (for data storage)
* Matplotlib (for visualization)

---

## 🧑‍💻 Usage

After running, a menu will appear:

```id="7o1lcn"
1. Add Expense
2. View Expenses
3. Monthly Summary
4. Category Analysis
5. Show Pie Chart
6. Exit
```

* Select options by entering numbers
* Provide required inputs when prompted

---

## 📊 Sample Output

* Displays total monthly expenses
* Shows category-wise breakdown
* Identifies highest spending category
* Provides suggestion to reduce spending
* Generates pie chart for visualization
